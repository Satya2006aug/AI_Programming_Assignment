# AI Programming Assignment
This repository contains implementations for the AI Programming assignment-2.

## Contents
1. Simple Reflex Agent for AQI monitoring
   - Group: Satya Vemula (SE24UCSE054), Aadya Kamani (SE24UCSE074), Kundana Reddy (SE24UCSE059)
   - Calculates AQI using pollutant data
   - Applies traffic restrictions based on AQI levels
3. Turing Test and CAPTCHA architecture design
4. BFS and DFS implementation for Tic Tac Toe

## Files
agent.py - AQI-based traffic control agent  
city_day.csv - dataset used for air quality data  
AQI_report.pdf - explanation of AQI logic  
tic_tac_toe_bfs_dfs.py - implemented BFS/DFS search methods on tic tac toe
Report_turing_CAPTCHA_bfs_dfs.pdf - report covering Turing Test, CAPTCHA, BFS, DFS

## How to Run
Run the AQI agent with: python3 agent.py       
Run the BFS/DFS with: python3 tic_tac_toe_bfs_dfs.py
